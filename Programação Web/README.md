## 🖥️ Programação Web

Esta disciplina foca na criação, desenvolvimento e manutenção de websites e aplicações web funcionais, englobando a parte visual (Frontend) e a arquitetura de interação e apresentação de dados.

Nesta disciplina, aprendi na teoria e na prática:

* **HTML:** Estruturação e organização de todo o conteúdo da página web.
* **CSS:** Aplicação de estilos, layout e design visual para tornar as páginas atraentes e responsivas.
* **JavaScript:** Implementação de interatividade, manipulação da DOM e lógica de comportamento do lado do cliente (Frontend).
* **Bootstrap:** Habilidade de aplicar este poderoso framework para agilizar o desenvolvimento.
